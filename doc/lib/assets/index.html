"use strict"

Sets up assets.

Parameters:

  • options must be an Object.

    • options.environment must be a String.
      (Should be either development or production.)

    • options.root must be a String.
      (The application root. Assets are resolved relative to this path.)

    • options.assetsBaseURI must be a String.
      (Assets are served from this URI.)

    • options.manifest must be an Object.
      (An object describing the assets. Format depends on whether you are running in development or production mode.)

Returns a Function

exports = module.exports = (options) -> {environment} = options if environment is 'development' return require('./development')(options) else if environment is 'production' return require('./production')(options) else throw new Error 'Invalid value for environment, should be one of `development` or `production`.' exports.task = require './task'
